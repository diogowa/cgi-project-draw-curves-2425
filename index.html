Curve Function: B-Spline

  • f: Change Curve Function
  • z: Stop Drawing Curve
  • C: Delete Curves
  • L: Hide Lines
  • P: Hide Segment Points
  • +/-: Add/Subtract Segment Points
  • SPACE: Pause animation
  • >/<: Faster/Slower Animation
  • 1: Enable Echos
  • T/t: Add/Subtract Echos
  • 2: Enable Gravity
  • 3: Enable Black&White
